# App Icon - "Patterns"

## Overview

The Behavior Tracker app icon features a neural network/behavioral pattern design that represents the interconnected nature of behavioral tracking and analysis.

## Design Elements

- **Color Scheme**: Blue to purple gradient (iOS blue #3A7BD5 to purple-blue #5856D6)
- **Pattern**: Neural network/node-based design with 6 outer circles connected to a central hub
- **Style**: Modern, clean, scientific/analytical
- **Theme**: Represents behavioral patterns, connections, and insights

## Icon Concept

The icon design symbolizes:
- **Central Hub**: The user/individual at the center
- **Six Outer Nodes**: The main behavioral pattern categories
- **Connecting Lines**: Relationships and correlations between patterns
- **Smaller Circles**: Sub-patterns and nuanced behavioral details
- **White on Blue**: Clarity and insight emerging from data

## How to Use in Xcode

### Method 1: Quick (Recommended)

1. Open your Xcode project
2. Navigate to `Assets.xcassets` in the Project Navigator
3. Select `AppIcon`
4. Drag `patterns-1024.png` into the **1024pt** slot
5. Xcode will automatically generate all other sizes

### Method 2: Manual (All Sizes)

1. Open your Xcode project
2. Navigate to `Assets.xcassets`
3. Select `AppIcon`
4. Drag each PNG file to its corresponding size slot:
   - patterns-1024.png → 1024pt
   - patterns-180.png → iPhone App iOS 60pt 3x
   - patterns-167.png → iPad Pro (12.9-inch) iOS 83.5pt 2x
   - patterns-152.png → iPad App iOS 76pt 2x
   - patterns-120.png → iPhone App iOS 60pt 2x
   - And so on for all sizes...

### Method 3: Replace AppIcon Set

1. Create a new AppIcon set in Assets.xcassets
2. Name it "AppIcon"
3. Import all patterns-*.png files
4. Xcode will match them by size automatically

## Customization

If you want to modify the icon:

1. **Edit SVG**: Open `patterns.svg` in a vector editor (Inkscape, Figma, etc.)
2. **Modify colors**: Change the gradient colors in the `<linearGradient>` section
3. **Adjust pattern**: Modify circle positions, sizes, or connections
4. **Regenerate**: Run `./generate_icons.sh` to create new PNG files

## Technical Details

- **Format**: PNG with transparency
- **Color Space**: sRGB
- **Corner Radius**: iOS standard (22.37% of icon size)
- **Generated by**: macOS qlmanage and sips
- **Source**: patterns.svg (scalable vector graphic)
